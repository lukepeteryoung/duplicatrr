Known Issues Movies:
- Bulk Deletion may delete movies that there is only 1 copy of, if it lists the same path twice. 
- False positives show sometimes without 2x folders. you can tell this because it return's a "[!] File path not found or unreachable on disk:" error.
- Extra or Special Episodes for TV Shows may be prioritised over the actual episode file.

Known issues TV Series:
- The scanner can sometimes mistake a subfoler with the same name as the actual file then flag as a duplicate, resulting in deletion.
